¡Claro! Te voy a proporcionar ejemplos de 6 solicitudes diferentes para usar en Postman. Estas incluirán:

2 solicitudes válidas (que pasen las validaciones y creen productos nuevos).
2 solicitudes con errores de validación (que generen un mensaje de error debido a campos faltantes).
2 solicitudes para actualizar productos ya existentes (suponiendo que el producto ya tiene un ID).
Aquí están los ejemplos:

Solicitudes Válidas
Producto 1 (Válido):
POST http://localhost:3000/products

json
Copiar código
{
    "title": "Smart TV Samsung 55\" 4K",
    "description": "Smart TV de 55 pulgadas, resolución 4K, con tecnología QLED.",
    "code": "SAMSUNG_4K_001",
    "price": 699.99,
    "status": true,
    "stock": 100,
    "category": "Electronics",
    "thumbnails": ["https://example.com/thumbnails/samsung_tv_55.jpg"]
}
Producto 2 (Válido):
POST http://localhost:3000/products

json
Copiar código
{
    "title": "Cámara Canon EOS 90D",
    "description": "Cámara DSLR de 32.5 MP, ideal para fotografía profesional.",

Solicitudes para Actualizar un Producto Existente
Producto 5 (Actualizar: Modificar precio y stock):
PUT http://localhost:3000/products/1
(Suponiendo que el producto con id: 1 existe)

json
Copiar código
{
    "title": "Smart TV Samsung 55\" 4K",
    "description": "Smart TV de 55 pulgadas, resolución 4K, con tecnología QLED.",
    "code": "SAMSUNG_4K_001",
    "price": 749.99,  // Modificado el precio
    "status": true,
    "stock": 120,  // Modificado el stock
    "category": "Electronics",
    "thumbnails": ["https://example.com/thumbnails/samsung_tv_55_new.jpg"]
}
Resultado esperado: El producto con id: 1 se actualizará con el nuevo precio y stock.
Producto 6 (Actualizar: Modificar solo el stock):
PUT http://localhost:3000/products/2
(Suponiendo que el producto con id: 2 existe)

json
Copiar código
{
    "title": "Cámara Canon EOS 90D",
    "description": "Cámara DSLR de 32.5 MP, ideal para fotografía profesional.",
    "code": "CANON_EOS_002",
    "price": 1249.99,
    "status": true,
    "stock": 30,  // Modificado el stock
    "category": "Cameras",
    "thumbnails": ["https://example.com/thumbnails/canon_eos_90d_updated.jpg"]
}
Resultado esperado: El producto con id: 2 se actualizará con el nuevo stock.
Resumen:
Productos válidos: El primero y segundo producto son correctos y se crearán sin problemas si se pasan correctamente los datos.
Errores de validación: El tercero y cuarto producto faltan campos obligatorios (price y status), por lo que recibirás un error indicando los campos faltantes.
Actualización de productos: El quinto y sexto producto son ejemplos de cómo actualizar un producto ya existente mediante una solicitud PUT usando su id.
Con estos ejemplos, puedes probar tanto la creación de nuevos productos como la actualización de productos existentes usando Postman.
